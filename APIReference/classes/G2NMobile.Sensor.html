startListen

Defined in api\G2NMobile.js:751

Syntax

startListen

(
  • delayOption
  • callback
)

Summary

This method is start the listener that detects the sensor value.

Caution!
If you are not call the addSensorChangedListener, you will receive the 'G2NSensorAlreadyStartListeningException' message.
If there nothing registered the event listener, you will receive the 'G2NSensorEventListenerException' message.

If wrong entered the G2NMobile.Sensor.Delay, you will receive the 'G2NWrongSensorDelayOptionException' message.


See Also : addSensorChangedListener, stopListen

Example: getSensor

Parameters:

  • delayOption Number

    FAST that bring changed sensor value very quickly. NORMAL that bring changed sensor value on medium speed.

  • callback Function
