G2NMobile.Sensor.SensorType

Defined in: api\G2NMobile.js:926
Module: G2NMobile

Summary

As SensorType class is a kind of type, this class indicate type of loaded sensor within Android-Phone. It is subordinate to the Sensor Class. And used in SensorManager class of G2NMobile module.
