G2NMobile.Wifi.WifiInfo

Summary

This method is used to create G2NMobile.Wifi.WifiInfo object. Input the index in first parameter of Wi-Fi list, for index about a AP of Wi-Fi list. Just obtain information for AP aboutG2NMobile.Wifi.WifiInfo/SSID:attribute, G2NMobile.Wifi.WifiInfo/level:attribute, G2NMobile.Wifi.WifiInfo/BSSID:attribute by this method.


Caution!
If Wi-Fi list dose not exist, you will receive 'G2NWifiListNullException',
If range of size of Wi-Fi list were out, you received 'G2NWrongWifiListIndexException' message.
